Features

Open Source

This app is completely Open Source. You can visit the source code on the website.

LibGDX

This app has LibGDX Java integration, so you can create LibGDX games.

Performance

The app is smooth and speedy on any Android device, enhancing the user experience.

Powerful

The app includes a great 3D model editor based on Jetpack Compose and uses the Sora Editor.
